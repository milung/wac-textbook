## Cvičenia s Angular a .NET Core

V tejto časti nájdete cvičenia pre predmet "Vývoj webových aplikácii v prostredí CLoud" pre rok 2020/2021. Cvičenia sú zamerané na vývoj webových aplikácii s využitím frameworku Angular a .NET Core. Ich obsah bol naposledy aktualizovaný vo februári 2021. Oproti aktuálnej verzii bol obsah viac zameraný na tvorbu webových aplikácií a menej sa venoval technikám cloud-native, a kubernetes. Technológie GitOps a micro Front-ends v tejto verzii ešte neboli obsiahnuté.

### Použité technológie:

* [Angular](https://angular.io/)
* [.NET Core](https://dotnet.microsoft.com/)
* [Docker](https://www.docker.com/)
* [Kubernetes](https://kubernetes.io/)
* [Azure](https://azure.microsoft.com/)
* [Azure DevOps](https://azure.microsoft.com/en-us/services/devops/)
* [Azure DevOps Pipelines](https://azure.microsoft.com/en-us/services/devops/pipelines/)
